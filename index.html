Projetos em Destaque

Monitoramento de Sinais Vitais (IoT)

Solução de IoT ponta a ponta desenvolvida para otimizar o cuidado em casas de repouso através de dispositivos vestíveis que monitoram sinais vitais em tempo real. Utilizando uma arquitetura de baixo consumo energético com protocolos BLE e LoRa, o sistema centraliza dados em uma plataforma web capaz de identificar anomalias personalizadas e disparar alertas automáticos, garantindo intervenções rápidas e maior segurança aos residentes.

  • Integração de hardware (ESP32) com software.
  • Lógica de detecção de anomalias em tempo real.
PHP Laravel MySQL C++ / ESP32
